# CutBuddy

## Introduction

This project is a Multi-Agent assistant for losing weight leveraging Claude Sonnet models for its agents and DuckDuck Go HTML Search and BeatifulSoup for meal planning. 

## Features

- **Multi-Agent System:** 3 unique specialized agents managed by an orchestrator agent that delegates tasks depending on user query.

- **Calorie Deficit Calculation:** Calculates TDEE and BMR based on user current weight, height, sex, and activity level.
  
- **Recipe Search:** Performs a recipe search based on ingredient preferences of the user using DuckDuck Go. 

- **Meal Plan Creation:** Creates a meal plan according to user weight loss goals and reccomended protein intake.


## Technologies Used:

- Strands SDK
- DuckDuck Go HTML Search
- Claude Sonnet Model 
- BeautifulSoup
